# Pillars of OOPS (C++ / Java) — Placement Exam Ready

These are core theory questions in MCQs, viva, and interviews.

---

## 1️⃣ Encapsulation

### Meaning

Binding data (variables) and methods (functions) into a single unit (class) and restricting direct access.

### How achieved

* Using classes
* Using access specifiers (`private`, `public`, `protected`)

---

## 2️⃣ Inheritance

### Meaning

One class acquires properties and behavior of another class.
